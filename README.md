# ProjetoCarometro
Este projeto consiste no desenvolvimento de um carômetro digital com interface gráfica em Java. O sistema permite cadastrar, buscar e visualizar estudantes, utilizando banco de dados MySQL e diretório para upload de fotos, garantindo organização e identificação visual dos alunos.
